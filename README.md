# API Data Retrieval and Storage
This project demonstrates how to retrieve data from a REST API, store it in a SQLite database, and display the stored records using Python.
For this project Open Library API has been used:"https://openlibrary.org/search.json?q=python"
The project fetches book information from the Open Library API and stores key attributes such as title, author, and publication year in a local SQLite database.

Through this project, I gained experience in:
- Consuming REST APIs using Python
- Working with JSON data
- Designing SQLite databases
- Performing CRUD operations
- Executing SQL queries for data retrieval
